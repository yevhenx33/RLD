## Core Terms

| Term | Meaning |
| --- | --- |
| RLD | Rate-Level Derivatives, the protocol category for trading interest-rate levels on-chain. |
| RLP | Rate-Level Perp, the perpetual tokenized rate exposure built by RLD. |
| `wRLP` | Wrapped Rate-Level Perp ERC-20 position token minted as debt and traded by longs. |
| Index price | Fundamental rate-derived value, usually `100 * borrowRate`. |
| Mark price | Market price of `wRLP` in the trading pool. |
| NF | Normalization factor, the global debt scaler for a market. |
| Debt principal | Account-level principal before NF scaling. |
| True debt | `debtPrincipal * normalizationFactor`. |
| Debt value | `trueDebt * indexPrice`. |
| Borrow rate | The underlying lending-pool rate tracked by a market. |
| TWAR | Time-weighted average rate. |
| TWAMM | Time-weighted average market maker, used for streaming orders and duration management. |
| CDP | Collateralized debt position. RLD shorts mint `wRLP` from collateral. |

## Actors

| Actor | Meaning |
| --- | --- |
| Long | Buys `wRLP` to benefit from rising rates. |
| Short | Mints `wRLP` debt and benefits from falling rates or favorable funding. |
| Minter | Another term for short. |
| LP | Provides liquidity to the `wRLP` / collateral pool. |
| Curator | Per-market risk manager allowed to propose/cancel timelocked risk updates. |
| Liquidator | Permissionless actor that closes insolvent positions for a bonus. |
| Settlement module | Per-market contract allowed to enter global settlement and invalidate withdrawal queues. |
| Underwriter | CDS short that provides collateral against protection demand. |
| Protection buyer | CDS long that wants payout when the referenced market enters stress. |

## Contracts

| Name | Meaning |
| --- | --- |
| `RLDCore` | Singleton protocol accounting contract. |
| `RLDStorage` | Storage and transient helper base for core. |
| `RLDMarketFactory` | Deploys complete market infrastructure. |
| `PrimeBrokerFactory` | Deploys user broker clones and mints ownership NFTs. |
| `PrimeBroker` | User-owned smart margin account. |
| `PrimeBrokerOpsModule` | Broker delegatecall module for heavy operations. |
| `PositionToken` | ERC-20 `wRLP` token for a market. |
| `BrokerRouter` | Periphery router for deposit, long, short, and close flows. |
| `BondFactory` | Periphery contract for creating and closing synthetic bonds. |
| `StandardFundingModel` | Funding model based on mark-index divergence. |
| `CDSDecayFundingModel` | CDS funding model based on fixed exponential decay. |
| `DutchLiquidationModule` | Health-based liquidation bonus and seize calculation. |
| `RLDAaveOracle` | Aave rate-to-index adapter. |
| `GhostSingletonOracle` | GhostRouter observation oracle. |
| `GhostRouter` | DEX router and observation layer. |
| `TwapEngine` | Streaming-order engine. |

## Parameters

| Parameter | Meaning |
| --- | --- |
| `K` | Scalar converting borrow rate to index price. |
| `minColRatio` | Required collateral ratio for new debt minting. |
| `maintenanceMargin` | Solvency threshold for liquidation. |
| `liquidationCloseFactor` | Max debt portion liquidatable while the account is not underwater. |
| `fundingPeriod` | Timescale for standard funding. |
| `badDebtPeriod` | Timescale for bad debt socialization. |
| `debtCap` | Max market debt in economic terms. |
| `minLiquidation` | Minimum liquidation size. |
| `liquidationParams` | Packed Dutch liquidation config. |
| `decayRateWad` | Annualized fixed decay rate for CDS markets. |
| `globalSettlementTimestamp` | Timestamp indicating terminal settlement mode. |

## Product Terms

| Term | Meaning |
| --- | --- |
| Synthetic bond | Short `wRLP` plus TWAMM unwind to fix floating yield over a duration. |
| Fixed-rate borrow | Long `wRLP` hedge against variable borrow cost increases. |
| Rate arbitrage | Multi-market trade that locks rate spreads. |
| CDS | Credit-default swap style protection using rate/utilization stress as trigger signal. |
| Parametric insurance | Payout model based on objective observed variables rather than claims governance. |
| Global settlement | Terminal market state used for CDS-style seizure and distribution flows. |
| Withdrawal queue | Broker delay mechanism for debt-bearing CDS collateral withdrawals. |
| Bad debt socialization | Gradual NF increase used to spread unbacked principal across remaining market debt. |
